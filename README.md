# ExoTRed
Exoplanetary Task Reduction, ExoTRed, is a python/pyraf package to reduce, calibrate and analysis data from observations of exoplanets.

___

Installation:

Under Ureka or astroconda enviromment, do:

```python
python setup.py install
```
